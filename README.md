# Tech Excuse Generator

A small browser-based tool for generating corporate tech excuses on demand. Hosted in https://techexcuse.com

## Files

- `index.html` — main app markup
- `style.css` — styling and responsive layout
- `app.js` — application logic and UI interactions
- `excuses.js` — excuse database used by the generator

## Usage

1. Open `index.html` in your browser.
2. Click `Generate Excuse` or press `Space`.
3. Copy or share the generated excuse.

## Notes

- The app uses a static excuse list in `excuses.js`.
- Counters in the UI are updated dynamically from the excuse array.
- Built for playful use only.
- This project is made for fun purposes only and is not intended for serious workplace use.
